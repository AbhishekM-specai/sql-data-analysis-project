# sql-data-analysis-project

## Overview
A collection of 25+ SQL queries written in MySQL to solve real-world 
business problems across datasets including customers, orders, 
products, and employees.

## Topics Covered
- SELECT, WHERE, DISTINCT, LIKE
- GROUP BY, HAVING, ORDER BY
- Aggregate Functions (MIN, MAX, AVG)
- CASE / IF Conditional Logic
- INNER JOIN, LEFT JOIN, Multi-table JOINs

## Tools Used
- MySQL
